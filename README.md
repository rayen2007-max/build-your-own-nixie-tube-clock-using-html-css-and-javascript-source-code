# Build Your Own Nixie Tube Clock

## Overview

This project allows you to create a digital representation of a Nixie tube clock using HTML, CSS, and JavaScript. The clock displays the current time in a visually appealing format that mimics the classic Nixie tube design.

## Features

- Real-time clock display in 12-hour format with AM/PM.
- Displays current date (month, day, and year).
- Responsive design for various devices.
- Simple and easy to customize.

## Technologies Used

- **HTML**: For the structure of the clock display.
- **CSS**: For styling the clock to resemble Nixie tubes.
- **JavaScript**: For updating the time and date dynamically.
